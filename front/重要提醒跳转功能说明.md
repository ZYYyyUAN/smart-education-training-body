# 重要提醒跳转功能说明

## 功能概述1111

重要提醒功能允许学生查看即将到期的作业和考试，并通过点击卡片直接跳转到相应的答题界面或考试确认界面。

## 功能特性1112111

### 1. 智能识别类型
- **作业类型**：显示橙色标签 "作业 · 即将截止"
- **考试类型**：显示红色标签 "考试 · 即将截止"

### 2. 详细信息显示

#### 作业卡片显示内容：
- 📚 课程名称（带渐变背景的标签）
- 📝 作业标题
- 📄 作业描述（如果没有描述则显示"暂无详细描述"）
- 📅 截止日期

#### 考试卡片显示内容：
- 📚 课程名称（带渐变背景的标签）
- 🧪 考试标题
- 📄 考试描述（如果没有描述则显示"暂无详细描述"）
- ⏱️ 考试时长（分钟）
- 📅 截止日期

### 3. 智能跳转
- **作业卡片**：点击后跳转到 `/homework/:id` 路由（作业答题界面）
- **考试卡片**：点击后跳转到 `/exam/confirm/:id` 路由（考试确认界面）

### 4. 响应式布局
- 卡片高度：280px（固定高度确保一致性）
- 网格布局：自动适应不同屏幕尺寸
- 悬停效果：鼠标悬停时卡片上浮并增加阴影

## 技术实现

### 1. 数据获取
```javascript
// 从后端API获取即将到期的作业和考试
const res = await axios.get('/api/xiaozhi/student/upcoming', {
  params: {
    studentId: 1,
    days: 3  // 3天内截止
  }
})
```

### 2. 数据处理
- 兼容旧版本API（仅返回作业数组）
- 支持新版本API（分别返回作业和考试）
- 自动补充课程名称信息

### 3. 路由跳转
```javascript
const handleImportantItemClick = (item) => {
  if (item.type === 'exam') {
    router.push({ name: 'ExamConfirm', params: { id: item.id } })
  } else {
    router.push({ name: 'homework', params: { id: item.id } })
  }
}
```

## 样式设计

### 1. 卡片样式
- 白色背景，圆角边框
- 悬停时上浮2px，增加阴影效果
- 内容采用flexbox布局，确保均匀分布

### 2. 标签样式
- 作业标签：橙色警告样式
- 考试标签：红色危险样式
- 课程标签：蓝紫色渐变背景

### 3. 响应式设计
- 移动端：单列布局
- 桌面端：网格布局，自动调整列数

## 使用说明

1. 学生登录后，在"我的学堂"页面可以看到重要提醒区域
2. 系统自动显示3天内即将到期的作业和考试
3. 点击任意卡片即可跳转到相应的答题界面
4. 作业直接进入答题界面，考试先进入确认界面

## 注意事项

1. 确保后端API返回正确的数据结构
2. 课程名称信息需要从已选课程中补充
3. 跳转前会验证项目ID的有效性
4. 支持错误处理和用户反馈
